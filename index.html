Atajos (Mis favoritos)

Para que VS me de sugerencias:

(ctrl + space)

Aumentar el tamaño de la pantalla o disminuirla :

ctrl + "+" ó "-"

Seleccionar varios renglones a la vez para editarlos:

alt(en los renglones que deseas) + click

Encontrar código mediante palabras clave :

ctrl + f

Duplicar lineas:

shift + alt + (boton arriba ó abajo)

Comentar una parte de código ó bloque de código :

alt + shift + a

Comentar un bloque de código (2da opción) :

ctrl + "}"

Ajustar nuestro código al tamaño de la pantalla eliminando la barra de navegación inferior:

alt + z

Abrir terminal :

ctrl + ñ

Hacer el simbolo Backtick " ` " :

alt + 9 + 6

Hacer el simbolo de Circunflejo " ^ " :

alt gr + (su simbolo abajo del signo "+" o al costado de la "ñ")

Desplegar menú de emoticones en Windows

Simbolo de Windows + "."

Para mover una linea !

ctrl + shift + la flecha de arriba o abajo

Emmet

El circunflejo " ^ " nos ayuda a subir un nodo una etiqueta por lo tanto la siguiente no estara dentro de la anterior etiqueta, sino, aparte.

Los parentesis " ( ) " sirven para agrupar

Para añadir ID's : " # " / para añadir Clases : " . "

Los corchetes sirven añadir atributos : " [ ] "

Tambien debemos aprovechar que algunas etiquetas tienen opciones!

El signo de dolar " $ " nos sirve para hacer numeraciones.

El signo de arroba " @ " en conjunto con el dolar " $ " nos permite elegir desde que número inciar la númeración.

Para añadir contenido/texto utilizamos las llaves " { } "
